## 🇬🇧 English

> **Just describe your idea in plain English. We'll build your custom AI agent in 30 seconds, complete with anti-fail guardrails and copy-paste code!**

Build Codex-ready AI agents from plain English — no coding required.

Codex Empower Zero Framework is a zero-code agent builder that transforms a simple idea into a complete, exportable agent package, including system prompts, guardrails, test cases, OpenAI-compatible JSON, `AGENTS.md`, and starter Python code.

### ✨ Features
1. **One-Line Agent Builder**: Type your idea, and the framework automatically architects the agent.
2. **Interactive Chat Sandbox**: Test your newly generated agent directly in the browser.
3. **Engineering-Grade Outputs**: Generates `system_prompt.md`, `test_cases.md`, `AGENTS.md` and more.
4. **Auto-Evaluator**: Built-in AI judge to score your agent's safety and boundaries.
5. **Multi-Language UI**: Fully supports English, Simplified Chinese, Traditional Chinese, and Japanese.

### 🚀 How to Deploy / Quickstart

#### Method 1: Local Deployment (For Beginners)
1. Ensure you have [Python 3.10+](https://www.python.org/downloads/) installed.
2. Clone or download this repository:
   ```bash
   git clone https://github.com/your-username/codex-empower-zero-framework.git
   cd codex-empower-zero-framework
   ```
3. Install the required lightweight dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the Streamlit application:
   ```bash
   streamlit run app.py
   ```
5. A browser window will open automatically (usually at `http://localhost:8501`).

#### Method 2: One-Click Cloud Deployment (Streamlit Community Cloud)
1. Push this repository to your own GitHub account.
2. Go to [share.streamlit.io](https://share.streamlit.io/) and log in with your GitHub account.
3. Click **New app**, select your repository, set the branch to `main`, and the Main file path to `app.py`.
4. Click **Deploy**. Your app will be live and accessible via a public URL!

无需任何代码基础，用大白话构建符合 Codex 和大模型规范的 AI 智能体。

Codex 零代码赋能框架（Codex Empower Zero Framework）是一个零代码智能体构建器。它能将你的简单想法转化为完整的、可导出的智能体工程包，包含：系统提示词、安全护栏、测试用例、配置文件、`AGENTS.md` 以及可直接运行的 Python 启动代码。

### ✨ 核心功能
1. **一句话生成器**：输入想法，框架自动为你完成智能体架构设计。
2. **沉浸式互动沙盒**：生成后无需离开页面，直接在内置沙盒中试聊你的专属智能体。
3. **工业级输出**：自动生成 `system_prompt.md`、`test_cases.md`、`AGENTS.md` 等全套开发资产。
4. **自动评估引擎**：内置 AI 裁判，为你的智能体打分并扫描防翻车护栏。
5. **全自动国际化**：完美支持简体中文、繁体中文、英文与日文，包括生成的提示词内容均可实现母语化。

### 🚀 小白部署与启动教程

#### 方法 1：本地一键运行（推荐小白使用）
1. 确保你的电脑安装了 [Python 3.10+](https://www.python.org/downloads/)。
2. 下载并解压本项目的源码，在文件夹内打开命令行/终端。
3. 安装极简的依赖包：
   ```bash
   pip install -r requirements.txt
   ```
4. 启动可视化面板：
   ```bash

   ```
5. 浏览器会自动弹出一个网页（默认 `http://localhost:8501`），输入你的 OpenAI API Key 即可开始使用！
